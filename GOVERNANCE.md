# Governance

Aleph is governed under the [BDFL](https://en.wikipedia.org/wiki/Benevolent_dictator_for_life)
model. All final decisions regarding the project are made by the project creator, [Khrysys](https://github.com/Khrysys).

Code Owners approval is required for all pull requests. Currently, Khrysys is
the sole Code Owner.
